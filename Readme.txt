La idea del proyecto es analizar las marcas y precios de los celulares en una página de venta de electrodomésticos (Para éste caso elegimos la  página de fravega)

Para la primeta parte en la recolección de los datos utilizamos de las librerias de Python Request (para solicitar los datos a la página) BeautifulSoup4 (para especificar la solicitud) y Pandas (para estructurar los datos en tablas)

En todo momento nos comunicamos via Whatsapp y estructuramos el proceso con Trello. Además actualizamos todos los procesos mediante Github.
